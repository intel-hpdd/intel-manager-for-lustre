chroma-diagnostics is a commandline command.  Run it with no arguments to produce a
 directory, in /var/log/, containing a collection of diagnostic files.

Install:
  $ cd chroma-diagnostics
  $ pip install .

Uninstall:
  $ pip uninstall chroma_diagnostics

Run:
  $ chroma-diagnostics -vh
